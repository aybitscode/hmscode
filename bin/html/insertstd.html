Insert Student Details


Steps:

1. Enter the student id

2. Enter student name

3. Select the course

4. You can keep the default fee or you can rewrite it

5. Select the year of join

6. Enter the amount from the old records or enter the amount to be paid

7. Click the default image to insert image or you can have default image


8. Select the image & click on open button

9. Click on submit to insert the data


10. The dialog box with message is show if the data is inserted successfully
